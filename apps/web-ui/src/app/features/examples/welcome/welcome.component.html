Welcome to AI-Nx-Starter

An AI-optimized Nx monorepo starter for full-stack application development with Angular frontend and NestJS backend.

📋 Prerequisites

MongoDB

Database for storing application data

Default: mongodb://localhost:27017

Redis

Session storage and caching

Default: redis://localhost:6379

🏗️ Technology Stack

  • Angular 19
  • NG-ZORRO (Ant Design)
  • RxJS
  • TypeScript
  • NestJS 11
  • TypeORM
  • MongoDB
  • Redis (Sessions)
  • Pino Logger

🔨 Key Features

Auto API Client Generation

Run npm run gen-api-client to automatically generate TypeScript API client from NestJS controllers.

AI Agent Rules

Custom rules in .windsurf/rules/ help AI agents understand project conventions and perform development tasks efficiently.

Nx Monorepo

Powerful build system with caching, parallel execution, and dependency graph management.

📁 Project Structure

ai-nx-starter/
├── apps/
│   ├── web-ui/          # Angular frontend application
│   └── web-server/      # NestJS backend application
├── packages/            # Shared libraries and utilities
│   ├── types/           # Shared TypeScript types
│   ├── api-client/      # Auto-generated API client
│   ├── data-access-layer/ # Database entities & services
│   └── backend-common/  # Shared backend utilities
├── documents/           # Technical documentation
├── scripts/             # Build and utility scripts
├── .windsurf/           # AI agent rules and configurations
│   └── rules/           # Project-specific AI rules
└── nx.json              # Nx workspace configuration
